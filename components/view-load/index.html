滚动加载 ​

​ 元素进入视图才显示渲染到浏览器页面上  ​

基础用法 ​

<script setup>
import { DhViewLoad } from "dh-vue-ui";
</script>

<template>
	<div style="height: 100vh; background-color: blue"></div>
	<client-only>
		<dh-view-load>
			<template #default="{ show }">
				<div v-if="show" style="height: 100vh; background-color: aqua">
					进入视图才会显示的元素
				</div>
			</template>
		</dh-view-load></client-only
	>
</template>

属性 ​

属性名类型默认值说明
isOnceBooleantrue只要一次有效监听
optionsObject{threshold: 0.5 }观察器的回调时机。

如果只想在能见度超过 50% 时检测，可以使用 0.5 的值。如果希望每次能见度超过 25% 时都执行回调，则需要指定数组 [0, 0.25, 0.5, 0.75, 1]。默认值为 0（这意味着只要有一个像素可见，回调就会运行）。值为 1.0 意味着在每个像素都可见之前，阈值不会被认为已通过。

事件 ​

事件名说明回调参数
callback有效触发后的回调{ isIntersecting, intersectionRatio }

插槽 ​

插槽名说明
default传参 show(Boolean,) 进入视图后，该值会为 true，

show 用来显示进入空间会才渲染到 dom 的元素
