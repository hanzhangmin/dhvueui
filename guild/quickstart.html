快开始 ​

安装 ​

pnpm i dh-vue-ui

同时在使用我们组件库的时候需要在tsconfig.json中配置types:["dh-vue-ui/lib/src/components"],否则会报错找不到组件

按需引入 ​

vue
<template>
	<div>
		<dh-button>按钮</dh-button>
	</div>
</template>

<script>
import { defineComponent } from "vue";
import { DhButton } from "dh-vue-ui";

export default defineComponent({
	components: {
		DhButton,
	},
});
</script>

全局引入 ​

js
import { createApp } from "vue";
import App from "./App.vue";
import DhUI from "dh-vue-ui";

const app = createApp(App);
app.use(DhUI);
app.mount("#app");

nuxt 项目引入 ​

在ssr 项目中，对于一些使用仅仅浏览器端才有的api组件，需要包上 client-only，否则会报错，比如 dh-scroll-load

vue
<template>
	<client-only>
		<dh-scroll-load ...></dh-scroll-load>
	</client-only>
</template>
